# Dashboard Overview

A responsive dashboard built to visualize gym and student data.

---

## Project Overview

This project displays key metrics about gyms, owners, and students using interactive charts and summary cards.

The main goal is to help administrators quickly understand performance and distribution.

---

## Goals & Target Users

### Goals

- Provide clear visual insights.
- Reduce cognitive load when reading data.
- Support decision-making.

### Target Users

- Gym administrators
- Platform managers
- Internal staff

---

## Design Decisions

### Layout

- Card-based layout for KPIs.
- Charts grouped by context.
- Filters placed in top-right.

### Data Presentation

- Bar charts for comparisons.
- Pie charts for proportions.
- Numeric KPIs for quick scanning.
- Charts are responsive and optimized for small screens.

---

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Recharts
- shadcn/ui

---

## Accessibility

Accessibility features include:

- ARIA labels for charts.
- Screen-reader-only descriptions.
- Keyboard-friendly components.

---

## Responsive Design

- Mobile-first approach.
- Breakpoints for table and chart optimization.
- Abbreviated labels for small screens.
